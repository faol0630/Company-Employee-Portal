

1) crear paquete auth

2) dentro de auth, crear la class SpringSecurityConfig

3) dentro de auth crear el paquete filters

4) dentro de filters crear la class JwtAuthFilter

5) en la class SpringSecurityConfig declaramos el siguiente atributo:

    @Autowired
    private AuthenticationConfiguration authenticationConfiguration;

6) en la misma class SpringSecurityConfig dentro del metodo filterChain, agregar la siguiente linea:

    .addFilter(new JwtAuthFilter(authenticationConfiguration.getAuthenticationManager()))

7) tambien en la class SpringSecurityConfig agregar el siguiente metodo para poder probar nuestra app:

        @Bean
        PasswordEncoder passwordEncoder(){
            return NoOpPasswordEncoder.getInstance();
        }

8) dentro de service o dentro de auth crear la class JpaUserDetailsService
